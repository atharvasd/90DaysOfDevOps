## 🚀 Recommended Learning Path

1. Complete the core Linux curriculum (Days 1-13).
2. Complete **Lab 01** and **Lab 02** to transition from a basic Linux user to an advanced system administrator.
3. Review the **Linux Security Best Practices** guide. This is mandatory reading before provisioning any public-facing EC2 instance or Droplet.
