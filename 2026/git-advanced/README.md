## 🚀 Recommended Learning Path

1. Complete the core Git curriculum (Days 22-28).
2. Complete **Lab 01** to learn how to catch errors locally before they even reach GitHub.
3. Complete **Lab 02** to learn how to recover lost work and safely scrub sensitive data from your repository history.
4. Review the **Git Security Best Practices** guide to learn how to protect the integrity of the codebase against supply-chain and insider threats.
