## How to Read These Diagrams

The diagrams in this module are generated using **Mermaid.js**. GitHub automatically renders these code blocks into beautiful flowcharts. If you are viewing this locally in a code editor like VS Code, you can install the `Markdown Preview Mermaid Support` extension to see the visualizations.
